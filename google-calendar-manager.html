⚙️ Setup Google API

Untuk menggunakan aplikasi ini, kamu perlu Client ID dari Google Cloud Console. Lakukan sekali saja, lalu simpan.

  1. Buka Google Cloud Console
  2. Buat project baru atau pilih yang sudah ada
  3. Aktifkan Google Calendar API
  4. Buat OAuth 2.0 Client ID (tipe: Web Application)
  5. Tambahkan origin file HTML ini ke Authorized JavaScript Origins
    (jika buka dari file lokal, gunakan: http://localhost atau deploy ke hosting)
  6. Salin Client ID dan paste di bawah
